# MELAcontrol Ink Test

## Overview

The ink test is a reliable testing device for the weekly control of sealing seams for tightness according to DIN EN ISO 11607-1 and ASTM F1929-12.

## Features

The ink required for the test is contained in a practical disposable bag. The MELAcontrol Ink Test is sealed in a transparent sterilization packaging. The ink is then squeezed out of the pouch inside the packaging. With the ink now on the sealed seams, the seams on all four sides of the package can be precisely inspected.

Irregularities, flaws or channels in the sealing seams quickly become visible. The ink test is particularly suitable for testing bags with side gussets.
