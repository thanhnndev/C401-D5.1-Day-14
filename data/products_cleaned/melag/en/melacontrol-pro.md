# MELAcontrol Pro

## Overview

The so-called HELIX is covered by the EN ISO 11140-6 European Standard. This is a test body for "Class B" steam sterilizers, which simulate the complicated requirements for sterilizing hollow instruments. This test body is required for batch monitoring of the processing of "critical B" instruments.

MELAcontrol Pro with 40 indicator strips is a durable, usable test body for the power-user. The test strips can be re-ordered in a refill package with 250 indicator strips.
